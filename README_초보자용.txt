마이로또노트 로또번호 자동수집 세트
====================================

이 폴더는 GitHub에 그대로 올리면 됩니다.

들어 있는 파일
1. data/lotto_results.json
   - 앱이 나중에 읽게 될 당첨번호 파일입니다.
   - 현재 1회부터 1232회까지 들어 있습니다.

2. scripts/update_lotto.py
   - 다음 회차를 동행복권 공식 사이트에서 찾아 JSON에 추가합니다.

3. .github/workflows/update_lotto.yml
   - 매주 토요일 밤과 일요일 아침에 위 프로그램을 자동 실행합니다.

4. requirements.txt
   - 자동수집 프로그램에 필요한 도구 목록입니다.

중요
- 폴더 이름 앞에 점(.)이 붙은 .github 폴더도 반드시 함께 올려야 합니다.
- JSON을 직접 편집할 필요는 없습니다.
- GitHub 저장소를 만든 뒤 Actions 권한을 한 번 설정해야 합니다.
- 앱 연결은 GitHub 주소가 확정된 다음 단계에서 진행합니다.
